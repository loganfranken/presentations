Media Query

@media (expression) {
   /* CSS */
}
@media (max-width: 600px) {
   .info {
      width: 50%;
   }   
}

If the viewport is currently 600px or less

@media (min-width: 300px) {
   .callout {
      float: left;
   }   
}

If the viewport is currently 300px or greater

@media (max-device-width: 400px) {
   .input {
      display: block;
	  width: 100%;
   }   
}

If the actual screen is currently 400px or less
